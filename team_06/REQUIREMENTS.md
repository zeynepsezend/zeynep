# Hypergraph library
Copy the folder from Google Drive:
Download the folder from
https://drive.google.com/drive/folders/1J2HjaR1LUUq3A7ev?usp=drive_link
and copy its contents to geolib.

Unblock the files:
After downloading, you may need to unblock the files to avoid Windows security warnings.
You can unblock all files in the folder using PowerShell:

```powershell
Get-ChildItem -Path 'C:\\geolib' -Recurse | Unblock-File
```
